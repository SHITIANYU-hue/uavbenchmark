# 17A × 4JD 统一模式定义

## 四层模式

```
.1  本体目录       "面对的世界里有什么"（全集/类型/词表）
.2  操作方法与边界   "SUT 怎么做 + 物理上能做什么"
.3  质量标准锚定    "什么算合格"（评分锚点）
.4  兜底等协议      "出问题怎么办"（处置+交接+通知+升级）
```

## 命名规则

- `.1` 统一以 **"目录"** 结尾 — 这是全集/词表，不是具体场景
- `.2` 统一为 **"XX机制与YY"** — 方法 + 物理限制合在一起
- `.3` 统一以 **"标准"** 结尾 — 评分锚点，Grader 用这个打分
- `.4` 统一以 **"协议"** 或 **"处置"** 结尾 — 出问题后的全套规则

## 与 A×L 的映射

```
.1 本体目录   →  L1 "能不能理解这个域"
.2 方法与边界  →  L2 "能不能按这个方法执行（受物理约束）"
.3 质量标准   →  L3 "能不能持续达到标准"
.4 兜底协议   →  L4 "出问题能不能自主处置"
```
